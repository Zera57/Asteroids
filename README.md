# Asteroids

Это один из моих первых проектов с ООП.

Игра написанна на C# с использованием .NET Framework.

### Требования

* .NET Framework 4.6.1

### Управление :
* Стрелки (Перемещение)
* Пробел (Стрельба)
* Esc (Выход)
